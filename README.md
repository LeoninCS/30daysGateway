## 🚀 快速开始
1. 克隆项目
```bash
git clone https://github.com/LeoninCS/30daysGateway.git
```
2. 进入项目目录
```bash
cd 30daysGateway/day0  #选择你要学习第几天的项目，此处为第0天
```
3. 安装依赖
```bash
go mod tidy
```
4. 运行项目
```bash
cd /home/leon/GoCode/30daysGateway/day1 #选择你要运行第几天的项目，此处为第1天
go run cmd/main.go
```
5. 测试项目
**按照项目相应天数的技术文档中测试项目**
